# Запуск всех циклов SCCM
## Версия, запускаемая на клиенте
Название скрипта: 
```
Run-All-SCCM-Cycles.ps1
```
Запускается с клиента, командой: 
```
powershell -ExecutionPolicy Bypass -File "C:\temp\Run-All-SCCM-Cycles.ps1"
```
## Версия, запускаемая на сервере
Рядом со скриптом должен быть создан текстовый файл clients.txt, в котором должны содержаться имена устройств или их ip адреса. <br>
Название скрипта:
```
Run-SCCM-FromServer.ps1
```
